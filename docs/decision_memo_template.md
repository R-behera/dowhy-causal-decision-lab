# Decision memo template

## Project

- Title: DoWhy Causal Decision Lab
- Domain: Causal Product Analytics

## Why this memo exists

Turn causal inference into a decision lab with assumption tracking, effect estimates, and stakeholder-friendly policy recommendations.

## Memo template

- Situation: what changed in the data or model output
- Decision required today
- Top evidence supporting the recommendation
- Main risk if the recommendation is wrong
- Suggested owner and next checkpoint

## Operator reminder

The best output is not just a score. It is a recommendation a business owner can understand, question, and act on quickly.
